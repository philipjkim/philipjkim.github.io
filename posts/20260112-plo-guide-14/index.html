PLO 가이드: #14 - PLO8의 기초와 전략 - 하이-로우의 세계

2026-01-12philipjkim
#poker  #omaha  #strategy  #PLO 

시리즈의 마지막 편에 오신 것을 환영합니다. 지금까지 PLO(Pot-Limit Omaha)의 모든 것을 다뤘습니다. 이제 완전히 새로운 차원의 오마하, PLO8 (Pot-Limit Omaha Hi-Lo) 을 소개합니다.

PLO8은 팟을 하이 핸드와 로우 핸드가 나눠 갖는 스플릿 팟 게임입니다. 같은 4장의 홀 카드를 사용하지만, 전략은 완전히 다릅니다. 하이만 노리면 절반만 가져가고, 로우만 노리면 역시 절반입니다. 스쿱(Scoop) - 하이와 로우 모두 이겨서 팟 전체를 가져가는 것 - 이 승리의 핵심입니다.

이번 편에서는 PLO8의 기본 규칙부터 스타팅 핸드 선택, 그리고 하이-로우를 동시에 노리는 전략까지 다룹니다. 시리즈를 마무리하며, 오마하 세계로의 여정을 완성합니다.

이 글은 Claude Opus 4.5 을 이용해 초안이 작성되었으며, 이후 퇴고를 거쳤습니다.

PLO8이란? 기본 규칙과 개념#

하이-로우 스플릿 (Hi-Lo Split)#

PLO8에서 팟은 두 부분으로 나뉩니다:

  • 하이(High): 가장 강한 전통적 포커 핸드 (스트레이트, 플러시, 풀하우스 등)
  • 로우(Low): 가장 약한 5장 카드 조합 (8 이하의 카드로만 구성)

팟 분배:

상황 하이 승자 로우 승자 팟 분배
하이와 로우 다른 플레이어 플레이어 A 플레이어 B 각각 50%
한 플레이어가 둘 다 승리 (스쿱) 플레이어 A 플레이어 A 100%
로우 없음 플레이어 A 없음 하이가 100%
하이 또는 로우 타이 타이 타이 해당 부분 분할

로우 핸드의 조건: “8-or-Better”#

PLO8의 공식 명칭은 Pot-Limit Omaha 8-or-Better입니다. 로우 핸드가 성립하려면 퀄리파이(Qualify) 조건을 충족해야 합니다.

로우 퀄리파이 조건:

규칙 설명
5장 모두 8 이하 9 이상의 카드가 포함되면 로우 불가
5장 모두 다른 숫자 페어가 있으면 로우 불가
에이스는 로우로 사용 가능 A는 1로 취급 가능
스트레이트/플러시는 로우에 영향 없음 A2345 플러시도 로우로 인정

로우 랭킹 (낮을수록 강함):

순위 로우 별명
1 A-2-3-4-5 휠 (Wheel), 넛 로우
2 A-2-3-4-6 6-4 로우
3 A-2-3-5-6 6-5 로우
4 A-2-3-4-7 7-4 로우
5 A-2-3-5-7 7-5 로우
… … …
최악 4-5-6-7-8 8-7 로우 (가장 약한 퀄리파이)

로우 비교 방법: 가장 높은 카드부터 비교합니다.

예: A-2-4-5-7 vs A-2-3-6-7

  • 둘 다 7이 가장 높음 → 다음 카드 비교
  • 첫 번째: 5, 두 번째: 6
  • 5 < 6 → 첫 번째 로우가 승리

보드 조건과 로우#

로우가 가능하려면 보드에 최소 3장의 8 이하 카드가 있어야 합니다.

보드 로우 가능 여부 이유
A♠3♥7♣K♦2♠ 가능 A, 3, 7, 2 = 4장의 로우 카드
A♠3♥7♣K♦Q♠ 가능 A, 3, 7 = 3장의 로우 카드
A♠3♥K♣Q♦J♠ 불가능 A, 3 = 2장뿐, 3장 미달
9♠T♥J♣Q♦K♠ 불가능 8 이하 카드 0장

같은 카드로 하이와 로우 만들기#

PLO8에서도 정확히 2장의 홀 카드를 사용해야 합니다. 하지만 하이와 로우에 다른 2장을 사용할 수 있습니다.

예시:

내 핸드: A♠2♣K♠Q♣ 보드: 3♠7♠8♥T♦J♦

  • 하이: K♠Q♣ 사용 → KQJT8 스트레이트? 아니, KQ로는 스트레이트 안 됨
  • 다시 계산: K♠Q♣ → A♠와 K♠로… 아니, 스페이드 3장이지만 플러시 안 됨

더 좋은 예시:

내 핸드: A♠2♣K♠3♣ 보드: 4♠5♠8♥T♦J♠

  • 하이: A♠K♠ 사용 → A♠K♠ + 4♠5♠J♠ = 넛 플러시
  • 로우: A♠2♣ 사용 → A-2-4-5-8 = 강한 로우

같은 핸드로 하이와 로우 모두 승리 → 스쿱!

스쿱의 중요성: PLO8 승리의 핵심#

왜 스쿱이 중요한가#

PLO8에서 팟의 절반만 가져가는 것은 장기적으로 수익적이지 않을 수 있습니다.

수학적 분석:

상황 결과 EV 영향
스쿱 팟 100% 획득 매우 +EV
하이만 승리 (로우 없음) 팟 100% 획득 +EV
하이 또는 로우만 승리 팟 50% 획득 종종 -EV 또는 0
쿼터드1 팟 25% 획득 거의 항상 -EV

쿼터드의 위험:

3명이 팟에 있고:

  • 플레이어 A: 넛 하이
  • 플레이어 B: 넛 로우
  • 플레이어 C: 넛 로우 (B와 같음)

팟 분배:

  • A: 50% (하이)
  • B: 25% (로우의 절반)
  • C: 25% (로우의 절반)

B와 C는 쿼터드 당했습니다. 팟에 넣은 돈 대비 25%만 받으면 대부분 손해입니다.

스쿱 가능한 핸드의 가치#

스쿱 잠재력이 높은 핸드:

핸드 유형 예시 스쿱 방법
A2 + 하이 카드 A♠2♣K♠K♥ 넛 로우 + 탑 세트/플러시
A2 + 수티드 A♠2♠3♥4♥ 넛 로우 + 플러시/휠
A3 더블 수티드 A♠3♠K♥Q♥ 세컨드 넛 로우 + 플러시/스트레이트
휠 카드 + 수트 A♠2♣3♠5♥ 휠 = 하이 스트레이트 + 넛 로우

휠(A2345)의 특별함:

휠은 PLO8에서 가장 강력한 핸드입니다:

  • 넛 로우 (A-2-3-4-5)
  • 5-하이 스트레이트 (하이로도 강함)
  • 두 방향 모두 강함 → 스쿱 가능성 극대화

로우만 노리는 것의 위험#

“로우만” 핸드의 문제:

문제 설명
최대 50% 이겨도 팟의 절반
쿼터드 위험 같은 로우와 분할
카운터페이트2 보드에 내 로우 카드가 페어되면 로우 약화
프리플랍 보드에 로우가 안 나올 수 있음

예시: 위험한 “로우만” 핸드

내 핸드: A♠2♣8♥9♦ (A2 있지만 하이 없음) 보드: 3♠5♥7♣K♦Q♠

  • 로우: A-2-3-5-7 = 넛 로우
  • 하이: 9-8-7-5-3? 아니, 2장만 사용해야 함 → 하이 없음

상대가 하이를 가지고 있으면 무조건 50%만 받습니다. 그리고 다른 플레이어도 A2를 가지고 있으면 쿼터드.

PLO8 스타팅 핸드 선택#

프리미엄 핸드#

티어 1: 최상위 핸드

핸드 특징 플레이
A♠A♥2♣3♦ AA + 넛 로우 드로우 항상 레이즈
A♠A♥2♣4♦ AA + 강한 로우 드로우 항상 레이즈
A♠2♣3♠4♥ 휠 드로우 + 수티드 에이스 레이즈/콜
A♠2♣K♠K♥ AA/KK + 넛 로우 드로우 레이즈

티어 2: 강한 핸드

핸드 특징 플레이
A♠3♣4♠5♥ 로우 드로우 + 휠 가능 레이즈/콜
A♠2♣Q♠J♥ 넛 로우 드로우 + 하이 카드 레이즈/콜
A♠A♥3♣4♦ AA + 세컨드 넛 로우 드로우 레이즈
2♠3♣4♠5♥ 휠 드로우 (A 없음) 콜/레이즈

플레이 가능한 핸드#

티어 3: 플레이 가능

핸드 특징 조건
A♠2♣7♥8♦ 넛 로우 드로우만 저렴한 팟에서
A♠3♣K♠Q♣ 세컨드 넛 로우 + 하이 포지션 있을 때
A♠4♣5♠6♥ 휠 가능 + 로우 멀티웨이
K♠K♥Q♠J♥ 하이 온리 프리미엄 선택적

피해야 할 핸드#

PLO8에서 약한 핸드:

핸드 유형 예시 문제점
미들 카드만 7♠8♣9♥T♦ 로우 없음, 하이도 약함
로우 없는 하이 K♠Q♣J♥T♦ 스쿱 어려움
약한 로우 드로우 A♠6♣7♥8♦ 카운터페이트 취약, 넛 아님
쓰레기 2♠7♣9♥K♦ 아무것도 없음
A 없는 로우만 2♠3♣4♥8♦ 로우 넛 못 만듦

핸드 선택 원칙#

PLO8 핸드 평가 기준:

요소 중요도 설명
A2 보유 최상 넛 로우 잠재력
A3 보유 높음 세컨드 넛 로우
수티드 에이스 높음 넛 플러시 잠재력
휠 카드 연결 높음 휠 스트레이트 가능
하이 온리 가치 중간 AA, KK + 보조
미들 카드 낮음 대부분 폴드

황금률: A2 또는 A3이 없는 핸드는 매우 선택적으로만 플레이

로우 드로우 플레이의 핵심#

넛 로우 드로우 vs 세컨드 넛 로우 드로우#

넛 로우 드로우 (A2):

보드 완성 시 로우 카운터페이트 위험
3-5-7 A-2-3-5-7 낮음
3-4-8 A-2-3-4-8 중간 (3 또는 4 페어)
4-5-6 A-2-4-5-6 낮음

세컨드 넛 로우 드로우 (A3):

보드 완성 시 로우 위험
2-5-7 A-2-3-5-7 누군가 A2 가지면 패배
4-5-8 A-3-4-5-8 누군가 A2 가지면 패배

핵심: A3은 보드에 2가 있을 때만 넛 로우가 됩니다.

카운터페이트 이해#

카운터페이트(Counterfeit) 는 보드에 내 로우 카드와 같은 카드가 페어되어 내 로우가 약해지는 것입니다.

예시:

내 핸드: A♠2♣K♥Q♥ 플랍: 3♠5♥8♣ 내 로우 드로우: A-2-3-5-8 (넛)

턴: 2♦

이제 보드: 3♠5♥8♣2♦

  • 내 로우: A-?-3-5-8 → 2가 보드에 있으므로 내 2를 쓸 수 없음
  • 새 로우: A-3-5-8-? → 한 장 부족, 내 홀 카드에서 가져와야 함
  • 내 다른 홀 카드: K, Q → 둘 다 8 이상이라 로우 불가!

카운터페이트 당함 → 로우 없음

카운터페이트 방지#

카운터페이트 방지 핸드:

핸드 카운터페이트 방지 이유
A♠2♣3♥4♦ 강함 2 또는 3이 페어되어도 다른 카드로 대체
A♠2♣3♥5♦ 강함 백업 로우 카드 있음
A♠2♣K♥Q♦ 약함 2 페어되면 로우 없음
A♠2♣3♥K♦ 중간 2 페어되면 A3으로 대체

핵심: 로우 카드 3장 이상 보유하면 카운터페이트에 강함

로우 드로우 플레이 전략#

플랍에서 로우 드로우:

상황 전략
넛 로우 드로우 + 하이 드로우 공격적, 레이즈 가능
넛 로우 드로우만 콜, 저렴하게
세컨드 넛 로우 드로우 주의, 멀티웨이에서 위험
로우 드로우 + 백업 콜, 안정적

메이드 로우:

상황 전략
넛 로우 + 하이 매우 공격적, 스쿱 추구
넛 로우만 밸류벳, 하지만 주의 (50%만)
세컨드 넛 로우 체크-콜 경향, 쿼터드 주의
로우 + 리드로우 공격적 가능

하이-로우 동시 추구 전략#

양방향 핸드의 파워#

양방향(Two-Way) 핸드는 하이와 로우 모두를 노릴 수 있는 핸드입니다.

양방향 핸드 예시:

핸드 하이 잠재력 로우 잠재력
A♠2♣3♠4♥ 휠, 플러시 넛 로우
A♠2♣K♠K♥ 탑 세트, 플러시 넛 로우
A♠A♥2♣3♦ 탑 세트 넛 로우
A♠2♣5♠6♥ 스트레이트, 플러시 넛 로우

포스트플랍 양방향 플레이#

플랍 예시:

내 핸드: A♠2♣K♠Q♣ 플랍: 3♠7♠8♦

분석:

  • 로우: A-2-3-7-8 = 넛 로우 드로우 (한 장 더 필요)
  • 하이: 넛 플러시 드로우 (A♠K♠)
  • 스트레이트: 없음

현재 상태:

  • 로우 드로우: 넛
  • 하이 드로우: 넛 플러시

전략: 매우 강함! 공격적으로 플레이. 레이즈/리레이즈 가능.

보드 텍스처별 전략#

로우 가능 보드 (3장+ 8 이하):

보드 고려사항
A-3-7 레인보우 로우 많음, 넛 로우/넛 하이 필요
2-4-K 투 톤 로우 가능, 하이에서 플러시 경쟁
3-5-6 연결 휠 가능, 스트레이트 주의

로우 불가능 보드 (2장 이하 8 이하):

보드 전략
K-Q-9 PLO처럼 플레이 (하이 온리)
A-K-T 로우 기대 어려움, 하이 집중
2-K-Q 턴/리버에 로우 올 수 있음, 대기

프리랩/백도어 로우#

프리랩(Freeroll): 상대와 같은 로우를 가지고 있지만, 하이로 이길 수 있는 상황.

예시:

내 핸드: A♠2♣K♠Q♠ 상대 핸드: A♥2♦7♣8♣ 보드: 3♠5♠8♦4♥7♠

  • 로우: 둘 다 A-2-3-4-5 = 타이
  • 하이: 내가 넛 플러시 (A♠K♠), 상대는 투 페어

결과:

  • 로우: 50% 분할
  • 하이: 내가 100%
  • 나는 75%, 상대는 25% (프리랩 성공)

프리랩 상황을 만들려면 하이 잠재력 있는 로우 핸드를 선택해야 합니다.

PLO vs PLO8: 핵심 차이점#

핸드 가치 변화#

핸드 PLO 가치 PLO8 가치 이유
A♠A♥K♠K♥ 최상 좋음 로우 없음, 스쿱 어려움
A♠2♣3♠4♥ 약함 최상 넛 로우 + 휠 가능
K♠Q♠J♥T♥ 프리미엄 약함 로우 전혀 없음
A♠A♥2♣3♦ 좋음 최상 AA + 넛 로우
7♠8♣9♥T♦ 플레이 가능 폴드 로우 없음, 하이도 너무 낮음

전략적 차이#

요소 PLO PLO8
핵심 목표 넛 하이 스쿱
핸드 선택 연결성, 수트 A2, A3, 양방향
블러프 빈도 있음 매우 낮음
팟 분할 없음 흔함
최고 핸드 더블 수티드 런다운 A2 + 하이 카드
미들 카드 가치 있음 거의 가치 없음

분산 차이#

요소 PLO PLO8
분산 높음 중간-높음
팟 분할로 인한 변동 없음 있음
스쿱 빈도 N/A 변수
쿼터드 위험 없음 있음

PLO8은 팟 분할로 인해 결과가 덜 극단적일 수 있지만, 쿼터드의 위험이 있어 주의가 필요합니다.

PLO8 실전 전략#

포지션의 중요성#

PLO8에서도 포지션은 매우 중요합니다.

포지션 전략
얼리 (UTG, MP) 프리미엄 핸드만, A2/A3 + 하이
미들 약간 넓게, 양방향 핸드
레이트 (CO, BTN) 가장 넓게, 선택적 스틸
블라인드 방어적, 팟 가격에 따라

멀티웨이 vs 헤즈업#

멀티웨이 팟 (PLO8에서 흔함):

전략 이유
넛 추구 여러 상대 중 누군가는 강함
블러프 금지 누군가는 콜함
스쿱 집중 50%만 받으면 이익 적음
프리미엄 핸드만 여러 상대 이겨야 함

헤즈업 팟:

전략 이유
더 공격적 상대 한 명만 이기면 됨
세컨드 넛도 가치 상대 넛 확률 낮음
블러프 가능 폴드 에퀴티 있음
50%도 의미 쿼터드 위험 낮음

흔한 실수와 해결책#

실수 1: 로우만 추구

실수 해결책
A2 있으면 무조건 플레이 하이 잠재력 확인
로우만 밸류벳 스쿱 가능할 때만 공격적

실수 2: 쿼터드 위험 무시

실수 해결책
같은 로우 가능성 무시 멀티웨이에서 A2 많음 주의
강하게 베팅 로우만 있으면 팟 컨트롤

실수 3: 카운터페이트 무시

실수 해결책
A2만 있으면 안전하다고 생각 백업 로우 카드 확인
카운터페이트 후에도 공격 새로운 로우 강도 재평가

실수 4: 하이 온리 핸드 과대평가

실수 해결책
PLO처럼 하이 온리 핸드 플레이 PLO8에서는 대부분 폴드
KQJT를 프리미엄으로 로우 없으면 절반만

PLO8 핸드 읽기#

상대 레인지 추정#

프리플랍 액션별 레인지:

액션 예상 레인지
림프 넓음, A2-A5, 스펙 핸드
레이즈 A2 + 하이, AA/KK + 로우
3bet 프리미엄 A2, AA + 로우
콜드 콜 A2-A4, 양방향 핸드

보드별 레인지 조정#

로우 완성 보드:

보드: 2♠5♥7♣A♦3♠

  • 상대 베팅: 로우 있음 (최소 A3 또는 24)
  • 상대 레이즈: 넛 로우 가능성 높음 (A2)
  • 상대 체크: 하이만 또는 약한 로우

로우 불가 보드:

보드: K♠Q♥J♣9♦T♠

  • 상대: PLO처럼 플레이, 스트레이트/플러시 레인지
  • 로우 고려 필요 없음

편 주제 핵심

행운을 빕니다. 테이블에서 뵙겠습니다! 🃏

  1. 쿼터드(Quartered): 팟의 25%만 받는 상황. 하이-로우 중 하나에서 이겼지만, 같은 부분에서 다른 플레이어와 타이가 되어 그 절반을 다시 나눠야 할 때 발생합니다. 예: 로우에서 이겼지만 다른 플레이어도 같은 넛 로우를 가지고 있으면, 로우 부분(50%)을 둘이 나눠서 25%씩 받습니다. ↩︎

  2. 카운터페이트(Counterfeit): 보드에 내 홀 카드와 같은 카드가 나와서 내 핸드가 약해지는 것. 특히 PLO8에서 로우 카드가 페어되면 내 로우가 무효화되거나 약해질 수 있습니다. 예: 내가 A2를 가지고 있는데 보드에 2가 페어되면, 더 이상 A2를 로우에 사용할 수 없습니다. ↩︎
